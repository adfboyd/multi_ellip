# KE-ratio density recurrence comparison

Generated by `studies/ke_ratio_density/analyze_recurrence_comparison.py`.

## Data sets

- `triaxial_nd2`: impulse BEM, shape `1:0.8:0.6`, `ndiv=2`.
- `triaxial_nd3`: impulse BEM, shape `1:0.8:0.6`, `ndiv=3`.
- `triaxial_exact`: analytic added-mass reference regenerated from the current `ndiv=2` inputs.
- `spheroid_nd2`: impulse BEM, shape `1:0.7:0.7`, `ndiv=2`.
- `spheroid_exact`: analytic added-mass reference for the spheroid inputs.

## Recurrence settings

- Observable: body-fixed orientation trajectory. Triaxial cases use the tracked
  marker point. Spheroidal cases use the antipodal axisymmetric-axis metric
  `acos(|a_i . a_j|)`, so the two signs of the symmetry axis are identified.
- Embedding dimension: `3`.
- Delay: `10` samples.
- Target recurrence rate: `0.03`.
- Theiler window: `50` samples.

## Main result

The spheroid cases are all classified as periodic for both impulse BEM and the
exact added-mass reference. No spheroid case changes behaviour class or coarse
behaviour family relative to the exact solution.

The triaxial cases are better described using an ensemble/regime metric rather
than only the per-run RQA label. Fixed-recurrence-rate RQA can keep a very high
determinism value for bounded deterministic chaos, so the script now reports
two chaos measures:

- `broadband_chaos_score`: short recurrence diagonals plus broad orientation
  spectrum.
- `chaos_score`: the broadband score plus short-time neighbour divergence in
  delay space.

On this basis, the low-density triaxial group is clearly separated from the
higher-density group:

- Exact triaxial, `rho=0.25`, `linKE/rotKE=20`: `5/5` runs are
  `chaotic-like`; grouped as `chaotic-regime`.
- Impulse triaxial `ndiv=2`, same parameters: `4/5` runs are
  `chaotic-like`; grouped as `chaotic-regime`.
- Impulse triaxial `ndiv=3`, same parameters: `4/5` runs are
  `chaotic-like`; grouped as `chaotic-regime`.
- Triaxial `rho=4`, `linKE/rotKE=1`: grouped as `regular-regime` for exact,
  `ndiv=2`, and `ndiv=3`. Some runs are labelled `sensitive-regular` because
  local neighbour divergence is large, but their spectra remain narrow and the
  grouped broadband score stays below the chaotic-regime threshold.

These labels should be read as finite-time recurrence diagnostics, not proof of
chaos. However, unlike the spheroid/axisymmetric cases, the single-body
triaxial ellipsoid dynamics are not generically integrable. The irregular exact
reference cases may therefore be physically meaningful rather than simply a
numerical artifact. The current recurrence result is strongest where the exact
triaxial reference itself loses the long clean diagonal bands seen in the
spheroid cases.
